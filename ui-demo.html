🎯 Hassbeam Connect - UI Demo

Benutzeroberfläche für IR-Code Aufnahme

📱 Hier würde die Hassbeam Connect Karte erscheinen

Diese Demo zeigt Ihnen, wie die UI aussehen würde

In Home Assistant würden Sie hier IR-Codes aufnehmen können

📋 So installieren Sie die UI:

  1. Kopieren Sie www/hassbeam-card.js in Ihr www/ Verzeichnis
  2. Registrieren Sie die Resource in Home Assistant
  3. Fügen Sie die Karte zu Ihrem Dashboard hinzu

1. Dateien überprüfen

✅ custom_components/hassbeam_connect/ - Integration vorhanden
✅ www/hassbeam-card.js - Frontend-Karte erstellt
⚠️ UI_INSTALLATION.md - Detaillierte Anleitung

2. Integration einrichten

In Home Assistant:

Einstellungen → Geräte & Dienste → + Integration hinzufügen → "Hassbeam Connect"

3. Frontend-Resource registrieren

Für HACS-Installation:

Einstellungen → Dashboards → Ressourcen → + Ressource hinzufügen URL: /hacsfiles/hassbeam-connect/hassbeam-card.js Typ: JavaScript Modul

Für manuelle Installation:

Einstellungen → Dashboards → Ressourcen → + Ressource hinzufügen URL: /local/hassbeam-card.js Typ: JavaScript Modul

4. Karte zum Dashboard hinzufügen

type: custom:hassbeam-connect-card

5. Features der UI

  • 🎯 IR-Code Aufnahme: Einfache Eingabe von Gerät und Aktion
  • 📋 Kürzliche Codes: Anzeige der letzten aufgenommenen Codes
  • 🔄 Live-Updates: Automatische Aktualisierung bei neuen Codes
  • ⚡ Services: Integration in Home Assistant Automatisierungen
  • 🎨 Modern Design: Responsive und benutzerfreundlich

6. Verfügbare Services

# IR-Code Aufnahme starten service: hassbeam_connect.start_listening data: device: "tv" action: "power" # Kürzliche Codes abrufen service: hassbeam_connect.get_recent_codes data: limit: 10

🚀 Nächste Schritte:

1. Folgen Sie der UI_INSTALLATION.md für die detaillierte Installation

2. Stellen Sie sicher, dass Ihr HassBeam-Gerät korrekt konfiguriert ist

3. Testen Sie die UI mit echten IR-Codes
